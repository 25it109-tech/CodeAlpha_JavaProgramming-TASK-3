# Task 3: Hotel Reservation System

## 📋 Project Description
A complete Hotel Room Booking System with multiple room types, booking management, and data persistence.

## ✨ Features
- Three room categories: Standard, Deluxe, Suite
- View available rooms
- Book rooms with customer details
- View all bookings
- Cancel bookings
- File handling (Save & Load bookings)
- Menu-driven console interface

## 🛠 Technologies Used
- Java (JDK 21)
- ArrayList
- File I/O (bookings.txt)
- OOP (Inheritance & Encapsulation)

## 🚀 How to Run
1. Open the project in VS Code
2. Compile: `javac HotelReservationSystem.java`
3. Run: `java HotelReservationSystem`

## 📌 Key Functionalities
- Room Booking System
- Real-time availability
- Booking Management
- Data Persistence
